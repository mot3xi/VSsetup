# Commands to see what the processor info is 

1. `free -h` - This si to see the free ram
2. `df -h ` - This is to see diskspace
3. `cat /proc/cpuinfo` - Processor details 

Easiest way is to install neofetch 

`sudo apt-get install neofetch`
